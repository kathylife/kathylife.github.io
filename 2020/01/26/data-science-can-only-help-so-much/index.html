    Data Science Can Only Help So Much

    January 26, 2020



    Data science is now a lot more mature than it was back in 2003 (the year when the SARS epidemic struck).

    Understandably, big data is already playing a big role in helping to fight infectious diseases like Ebola and polio.

    How can we leverage data science advancements to speed up the eradication of the latest coronavirus outbreak? That sure is giving scientists some food for thought this week.

    That being said, no predictive model — no matter how accurate — is going to keep us from harm’s way if policymakers are not stepping up to take proper measures to protect the people.
